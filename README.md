# Projeto-de-Sistema-de-Academia

Projeto de Programação do curso de Ciências da Computação na Universidade de BrazCubas - Mogi Das Cruzes, Período Matutino

Sobre o Projeto

Este projeto foi desenvolvido em Python com o objetivo de auxiliar na administração de uma academia. O sistema permite cadastrar alunos, organizar atendimentos na recepção, controlar pagamentos e registrar avaliações físicas de acordo com o plano escolhido por cada aluno.

Entre as principais funcionalidades estão o cadastro de alunos com suas informações pessoais e modalidade esportiva, a listagem dos alunos cadastrados, a atualização do status de pagamento e o controle de alunos com pendências financeiras.

O sistema também possui uma fila de atendimento para a recepção, seguindo a ordem de chegada dos clientes, além do controle das avaliações físicas disponíveis em cada plano (Básico, Gold e Black), registrando as sessões realizadas por cada aluno.

Durante o desenvolvimento foram utilizados conceitos importantes de programação, como listas, dicionários, funções, tratamento de erros, modularização do código e estrutura de filas, proporcionando uma aplicação prática dos conteúdos estudados em sala de aula.
